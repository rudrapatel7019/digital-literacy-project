# Task 1 – Digital Literacy Awareness Infographic

## Tool Used
**Canva** (canva.com) – Free online design platform

## What the Infographic Shows

The infographic is titled **"Be Digitally Smart!"** and covers three core areas:

1. **What is Digital Literacy?**  
   Defined as the ability to use digital tools effectively, safely, and responsibly. Illustrated with icons for devices, the internet, and communication.

2. **Useful Digital Tools for Students**  
   Highlights Google Workspace (Docs, Sheets, Forms), GitHub for version control, Canva for design, and HackerRank for coding practice.

3. **Safe Internet Practices**  
   Covers tips like using strong passwords, not sharing OTPs, identifying phishing links, and reporting cybercrime at cybercrime.gov.in.

## Design Approach
- Used a blue and white colour palette for a clean, professional look.
- Three-column layout with icons for visual clarity.
- Designed to be readable on a mobile screen as well as projected on a classroom screen.
